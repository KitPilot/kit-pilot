# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its code of
conduct. The full text is published at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project — whether as a maintainer, contributor,
issue reporter, or user — you agree to abide by the terms set out in that
document.

## Scope

The Code of Conduct applies to all project spaces, including this
repository, its issue tracker, pull requests, discussions, and any
project-affiliated communication channels. It also applies when an
individual is representing the project in public spaces.

## Enforcement

Concerns about contributor behavior in any of the above spaces can be
reported privately to the maintainers via the contact channel listed in
[SECURITY.md](./SECURITY.md). Reports will be reviewed and handled
confidentially. Maintainers may take any action they deem appropriate,
including warning, suspension, or removal of access to the project, in
order to keep the community welcoming and productive.
